Pour que le projet soit fonctionnel après un import de Git :
1. dans un terminal : composer install
2. copier/coller le fichier ".env.example"
3. renommer la copie en ".env"
4. vérifier la ligne "DB_DATABASE" contienne la base de données "projet_web2"

Pour avoir la base de données :
1. ouvrir xampp et créer une base de données du nom de "projet_web2"
2. dans un terminal : php artisan migrate:fresh --seed

Documentation sur les components

Créer un utilisateur : a besoin d'un $user, reçu via le return view('', ['user' => auth()->user()]) ou valeur nulle
<x-create_user :admin="$user->admin"></x-create_user>

Supprimer un utilisateur : a besoin de recevoir l'id de l'utilisateur
<a href="delete-user/{id}">Supprimer l'utilisateur</a>

Formulaire de connexion :
<x-login></x-login>
